This directory contains optimized images generated by the image optimization script.

The script generates multiple formats and sizes for each source image:

1. For each image in src/app/img (except logo.jpeg which is preserved as-is):
   - Multiple sizes: 320px, 640px, 1024px, 1920px (width)
   - Multiple formats: JPEG, WebP, AVIF
   - Placeholder thumbnails for blur-up loading effect

Example structure:
- clean1-320.jpg
- clean1-320.webp
- clean1-320.avif
- clean1-640.jpg
- clean1-640.webp
- clean1-640.avif
- clean1-1024.jpg
- clean1-1024.webp
- clean1-1024.avif
- clean1-1920.jpg
- clean1-1920.webp
- clean1-1920.avif
- clean1-placeholder.jpg

The OptimizedImage component automatically selects the most appropriate image based on:
- Device screen size
- Browser support for image formats
- Network conditions
- Viewport visibility (for lazy loading)
